# TS Calculator

A minimal TypeScript command-line calculator.

## Setup

```bash
npm install
```

## Build

```bash
npm run build
```

## Run

Pass the expression as a command-line argument:

```bash
npm start -- "1 + 2 * 3"
```

Expected output:

```text
7
```

Another example:

```bash
npm start -- "(10 - 3) / 2"
```

Expected output:

```text
3.5
```
